# SkiaViewer

**A hardened Silk.NET + SkiaSharp OpenGL viewer for .NET.**

SkiaViewer renders SkiaSharp raster surfaces to an OpenGL-backed window via texture upload.
It provides thread-safe lifecycle management, cross-thread shutdown, and frame-level exception recovery.
